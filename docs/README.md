# Ortie docs

Living design documents and plans for Ortie. The repository architecture itself is documented in the src/main.rs header, the same way lib.rs documents io-oauth.

[oauth21-plan.md](./oauth21-plan.md) tracks the OAuth 2.1 readiness and multi-grant work: the settled design decisions (grant selection, pkce shape, extras passthrough, wizard write-back), the milestone sequence (device grant, discovery upgrades, dynamic client registration, revocation) and the landed history.
